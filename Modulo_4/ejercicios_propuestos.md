1) Genera el codigo necesario para gestionar la solicitud de un pedido en McDonald's, permitiendo al cliente seleccionar la comida, la bebida y el postre. Como funcionalidad adicional, el sistema debe permitirle agrandar su pedido.
    Entrada:
        1- Mostrar el menu al usuario
        2- Solicitar al usuario que seleccione su comida, bebida y postre
        3- Preguntar al usuario si desea agrandar su pedido
    Proceso:
        1- Validar las opciones seleccionadas por el usuario
        2- Calcular el total del pedido
        3- Si el usuario desea agrandar su pedido, agregar el costo adicional al total
    Salida:
        1- Mostrar el resumen del pedido al usuario, incluyendo los elementos seleccionados y el total a pagar

2) Genera el codigo necesario para gestionar la solicitud de un envio de un paquete, permitiendo al cliente seleccionar el tipo de envío, la dirección de destino y el peso del paquete. Como funcionalidad adicional, el sistema debe permitirle agregar servicios adicionales como seguro o seguimiento.
    Entrada:
        1- Solicitar al usuario que seleccione el tipo de envío (por ejemplo, estándar, express, etc.)
        2- Solicitar al usuario que ingrese la dirección de destino
        3- Solicitar al usuario que ingrese el peso del paquete
        4- Preguntar al usuario si desea agregar servicios adicionales como seguro o seguimiento
    Proceso:
        1- Validar las opciones seleccionadas por el usuario
        2- Calcular el costo del envío en función del tipo de envío, el peso del paquete y los servicios adicionales seleccionados
    Salida:
        1- Mostrar el resumen del envío al usuario, incluyendo el tipo de envío, la dirección de destino, el peso del paquete, los servicios adicionales seleccionados y el costo total del envío

3) Genera el codigo necesario para gestionar la solicitud de una reserva en un restaurante, permitiendo al cliente seleccionar la fecha, la hora y el número de personas. Como funcionalidad adicional, el sistema debe permitirle agregar solicitudes especiales como una mesa cerca de la ventana o un menú vegetariano.
    Entrada:
    1- Solicitar al usuario que ingrese la fecha de la reserva
    2- Solicitar al usuario que ingrese la hora de la reserva
    3- Solicitar al usuario que ingrese el número de personas para la reserva
    4- Preguntar al usuario si desea agregar solicitudes especiales como una mesa cerca de la ventana o un menú vegetariano
    Proceso:
    1- Validar las opciones seleccionadas por el usuario
    2- Verificar la disponibilidad del restaurante para la fecha y hora seleccionadas
    3- Confirmar la reserva y las solicitudes especiales con el usuario
    Salida:
    1- Mostrar el resumen de la reserva al usuario, incluyendo la fecha, hora, número de personas y las solicitudes especiales seleccionadas

4) Genera el codigo necesario para gestionar la solicitud del pago del permiso de circulación de un vehículo, permitiendo al cliente seleccionar el tipo de vehículo, la fecha de vencimiento y el método de pago. Como funcionalidad adicional, el sistema debe permitirle agregar servicios adicionales como recordatorios de vencimiento o descuentos por pago anticipado.
    Entrada:
    1- Solicitar al usuario que seleccione el tipo de vehículo (por ejemplo, automóvil, motocicleta, etc.)
    2- Solicitar al usuario que ingrese la fecha de vencimiento del permiso de circulación
    3- Solicitar al usuario que seleccione el método de pago (por ejemplo, tarjeta de crédito, transferencia bancaria, etc.)
    4- Preguntar al usuario si desea agregar servicios adicionales como recordatorios de vencimiento o descuentos por pago anticipado
    Proceso:
    1- Validar las opciones seleccionadas por el usuario
    2- Calcular el costo del permiso de circulación en función del tipo de vehículo, la fecha de vencimiento y los servicios adicionales seleccionados
    3- Procesar el pago utilizando el método de pago seleccionado por el usuario
    Salida:
    1- Mostrar el resumen del pago al usuario, incluyendo el tipo de vehículo, la fecha de vencimiento, el método de pago, los servicios adicionales seleccionados y el costo total del permiso de circulación
