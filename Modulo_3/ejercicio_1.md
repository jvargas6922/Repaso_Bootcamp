1) Genera el codigo necesario para gestionar la solicitud de un pedido en McDonald's, permitiendo al cliente seleccionar la comida, la bebida y el postre. Como funcionalidad adicional, el sistema debe permitirle agrandar su pedido.

2) Genera el codigo necesario para gestionar la solicitud de un envio de un paquete, permitiendo al cliente seleccionar el tipo de envío, la dirección de destino y el peso del paquete. Como funcionalidad adicional, el sistema debe permitirle agregar servicios adicionales como seguro o seguimiento.

3) Genera el codigo necesario para gestionar la solicitud de una reserva en un restaurante, permitiendo al cliente seleccionar la fecha, la hora y el número de personas. Como funcionalidad adicional, el sistema debe permitirle agregar solicitudes especiales como una mesa cerca de la ventana o un menú vegetariano.

4) Genera el codigo necesario para gestionar la solicitud del pago del permiso de circulación de un vehículo, permitiendo al cliente seleccionar el tipo de vehículo, la fecha de vencimiento y el método de pago. Como funcionalidad adicional, el sistema debe permitirle agregar servicios adicionales como recordatorios de vencimiento o descuentos por pago anticipado.
